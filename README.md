# Go Learning Examples

Коллекция учебных примеров на Go, демонстрирующих различные концепции и паттерны программирования.

## Описание проекта

Этот репозиторий содержит набор практических примеров на языке Go, охватывающих различные темы от базовых концепций до продвинутых техник работы с конкурентностью, HTTP-запросами и обработкой данных.

## Содержание

### Работа с конкурентностью

- **conveyor.go** - Конвейерная обработка данных с использованием каналов
  - Генерация случайных слов
  - Фильтрация уникальных слов
  - Реверс строк
  - Объединение каналов (merge)

- **throttle.go** - Ограничитель скорости выполнения функций
  - Контроль частоты вызовов (rate limiting)
  - Использование тикеров и каналов

- **contextConveyor.go** - Работа с контекстом в конвейерах
- **contextTraining.go** - Практика использования context

### Игровая логика и обработка ошибок

- **game.go** - Текстовая игра с системой команд и объектов
  - Кастомные типы ошибок
  - Валидация действий игрока
  - Управление инвентарем

- **dontPanic.go** - Обработка паник и восстановление

### HTTP и сетевые запросы

- **httpGet.go** - HTTP GET-запросы с обработкой ошибок
  - Работа с заголовками и параметрами
  - Парсинг JSON-ответов
  - Обработка HTTP-статусов

- **httpHelper.go** - Вспомогательные функции для HTTP
- **fakeServer.go** - Тестовый HTTP-сервер

### Обработка данных

- **xmlToCsv.go** - Конвертация XML в CSV
  - Парсинг XML-структур
  - Запись CSV-файлов
  - Работа с вложенными структурами

- **parsepages.go** - Парсинг журнала задач
  - Работа с датами и временем
  - Регулярные выражения
  - Агрегация данных

### Работа с базами данных

- **SQLMap.go** - Реализация key-value хранилища на основе SQL
- **main.go** - Содержит SQLMap и другие утилиты

### Утилиты

- **slugify.go** - Преобразование строк в URL-friendly формат

## Запуск примеров

Каждый файл содержит функцию `main()` и может быть запущен отдельно:

```bash
go run game.go
go run conveyor.go
go run httpGet.go
# и т.д.
```

## Требования

- Go 1.18 или выше (используются generics и современные возможности языка)

## Особенности реализации

- Использование каналов для конкурентной обработки
- Кастомные типы ошибок с методом `Error()`
- Работа с `context` для управления жизненным циклом горутин
- Практика идиоматичного Go-кода
- Обработка ошибок через `errors.As()` и `errors.Unwrap()`

## Структура кода

Код организован по принципу "один файл - одна концепция", что упрощает изучение отдельных тем. Каждый пример содержит:
- Описание задачи в комментариях
- Реализацию решения
- Функцию `main()` с демонстрацией работы

## Лицензия

Учебный проект для изучения Go.
